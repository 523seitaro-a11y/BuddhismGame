# 初詣作法ならべ

スマホで遊べる、お寺の初詣の作法を学ぶドラッグ式ミニゲームです。

## 遊ぶURL

[https://523seitaro-a11y.github.io/BuddhismGame/](https://523seitaro-a11y.github.io/BuddhismGame/)

## 遊び方

1. 画面下の作法パネルを選びます。
2. 中央の枠へ、左から右の順番でドラッグして入れます。
3. スマホではパネルをタップしてから枠をタップしても操作できます。
4. 4枚入れ終わったら `決定` を押します。
5. 上部の人が並べた手順どおりに動き、最後に正解が表示されます。

## 正しい流れ

お賽銭、鈴を鳴らす、合掌、礼。
