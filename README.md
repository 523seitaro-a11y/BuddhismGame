# BuddhismGame

仏教の考え方を、短い選択式の体験ミニゲームとして学べるWebアプリです。

## 遊び方

1. `index.html` をブラウザで開きます。
2. QRコードで共有したい場合は、公開後のURLをQRコード化してスマホで読み取ります。
3. 呼吸、観察、選択、ふりかえりのステージを進めて、四つの学びを集めます。

## 公開の流れ

GitHub Pagesで公開すると、QRコードから手軽に遊べます。

1. GitHubにpushします。
2. リポジトリの `Settings` -> `Pages` を開きます。
3. `Deploy from a branch` で `main` / `/root` を選びます。
4. 公開URLをQRコード化します。

## ファイル構成

- `index.html`: 画面構造
- `styles.css`: 見た目とスマホ対応
- `app.js`: ゲーム進行
